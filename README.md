#  Projet : Compteurs Interactifs  
TP Réseaux – L3 SPI  
Auteur : momo12235554  

Ce projet consiste à développer un système de compteur interactif avec trois modes différents :  

1. **Version simple (main)** – incrémentation locale du compteur  
2. **Version Polling (Q4)** – récupération périodique de la valeur  
3. **Version WebSocket (Q5)** – mise à jour en temps réel  

Le serveur est développé avec FastAPI et les clients sont des pages web HTML/JS.
